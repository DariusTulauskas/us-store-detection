# US Store Detector Privacy Policy

**Effective date:** February 24, 2026

US Store Detector helps users identify shopping websites that may not ship from the United States.

## What this extension does

- Analyzes content on the shopping pages you visit.
- May analyze related same-site policy pages (for example: shipping, returns, contact) to improve detection accuracy.

## Data we process

- **Website content** from pages where the extension runs.
- **Host/domain information** needed to cache verdicts per site for performance.

## Data we store

The extension stores the following in Chrome storage:

- User settings (for example: thresholds and warning preferences).
- Domain override preferences (for example: mark a domain as `us` or `non_us`).
- Cached scan results (status, score, and reasons) keyed by domain.

## Data sharing and selling

- We do **not** sell user data.
- We do **not** transfer user data to third parties for unrelated purposes.
- We do **not** use user data for creditworthiness or lending decisions.

## Remote code

This extension does **not** use remote code execution.

## Security

Data is stored using Chrome extension storage APIs. No account login is required.

This policy may be updated when extension behavior changes. The updated policy will be posted at this URL.
